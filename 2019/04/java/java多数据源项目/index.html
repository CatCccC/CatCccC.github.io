多数据源与单数据源的对比

单数据源:

  1. 配置简单
  2. 事务管理清晰

多数据源:

  1. 配置繁琐 ,每一个数据源都需要重复 DataSource,DataSourceTransactionManager,SqlSessionFactory
  2. 每个事务管理 都需要指定对应的SqlSessionFactory
  3. 多个数据源事务 无法嵌套提交,一致性的问题 可参加 分布式事务的 最终一致性 相关的解决方法
